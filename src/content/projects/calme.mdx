## Project Overview

CALMe is a team-built hackathon project from Code for Resilience, providing immediate mental health support in crisis situations. The offline-first PWA guides users through trauma-informed conversations based on the Maa'se Model (מחויבות, עשייה, שאלות, הבנייה) to prevent PTSD, along with calming activities accessible without network connectivity. [lichyot](https://lichyot.org/model-maase/)

## My Role

As Co-founder and Team Lead, I coordinated the team including UX/UI architect, therapeutic content developer, resilience expert, and DevOps engineer. I shaped the product vision, designed the chat interface following the Maa'se Model, and oversaw offline data storage implementation. [prezi](https://prezi.com/p/e9kibihm8hgw/calme/)

## How It Works

Users interact with a natural chat interface that follows the four stages of the Maa'se Model: building commitment, encouraging action, asking concrete questions to shift to rational thinking, and constructing safety. Calming activities like breathing exercises are available offline via service workers and IndexedDB. [lichyot](https://lichyot.org/model-maase/)

The app ensures all content loads instantly in crises, delivering psychological first aid to civilians experiencing acute stress responses. [codeforresilience](https://www.codeforresilience.com/)

## Technical Challenges

A key challenge was implementing a conversational UI that adheres to the Maa'se Model's psychological structure while feeling intuitive and natural for users in distress. Ensuring full offline functionality required optimizing service workers, caching trauma-informed scripts, and handling data persistence without backend reliance. [prezi](https://prezi.com/p/e9kibihm8hgw/calme/)

We balanced academic fidelity with user experience, adapting Hebrew-originated model stages for global accessibility in a PWA format. [lichyot](https://lichyot.org/model-maase/)

## Tech Stack
Progressive Web App (PWA), Service Workers, IndexedDB, JavaScript, Trauma-Informed Design, Maa'se Model Implementation

## Impact

CALMe empowers civilians with immediate, evidence-based tools to mitigate PTSD risk during traumatic events, bridging gaps in mental health access where connectivity fails. Built for real-world resilience, it supports acute stress response intervention within the critical first 48 hours. [codeforresilience](https://www.codeforresilience.com/)
