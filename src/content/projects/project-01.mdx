## Project Overview

Our team developed an offline-first progressive web app that provides immediate, accessible mental health support during crisis situations. Built for civilians experiencing distress, it delivers trauma-informed conversations and calming activities without requiring network connectivity.

## Technical Challenges

One of the biggest challenges was creating a chat like interface that followed the psychological and academic outline of using the Maa'se Model  for PTSD prevention, while allowing users to interact with the app in a natural way. We also had to ensure that all content was available offline, which required careful planning of data storage and retrieval.
